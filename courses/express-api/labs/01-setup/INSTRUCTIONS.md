# Setup & CRUD

Build a REST API with Express and Postgres.

## Tasks

- Connect to the database
- Create the users table
- Implement CRUD endpoints
- Run the seed script
- Verify all tests pass

## Hints

The database connection is in `src/db.js`. The server is in `src/index.js`. Run `node seed.js` to populate initial data.

Use `Run Tests` in the testing panel to check your progress.
